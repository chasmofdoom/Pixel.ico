This repository includes pixel art 16x16 icons for most of the apps I use. The ICO files are upscaled to 256x256 to stop them from looking blurry. The PNG files are 16x16 for easier modification.<br>
To use custom icons on the desktop, create a shortcut to the application, right click on the shortcut, select "properties", and select "Change Icon". Then, click "Browse", and select an ICO file.
For the trash bin and user icons, go to "Settings>Personalization>Themes>Desktop Icon Settings", select the application you want to change, and click "Change Icon".
